# Diabetes-Prediction-Web-Application
This project is an end-to-end deep learning system for diabetes status prediction using clinical and biochemical patient data. It integrates model training, API inference, prediction logging, and interactive visualization. Trained using a deep learning neural network (Keras/TensorFlow)

Dataset Features include:

Gender

Age

Urea

Creatinine

HbA1c

Cholesterol

Triglycerides

HDL, LDL, VLDL

BMI

Output classes:

N – Non-Diabetic

P – Pre-Diabetic

Y – Diabetic
